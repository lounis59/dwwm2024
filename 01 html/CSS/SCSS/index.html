Les base de SASS et de SCSS

  • Pour installer SASS in utilisera
    npm install -g sass
  • Pour compiler du sass-scss en css on utilisera la commande
    sass cheminVerFichierSource.scss cheminVerFichierSource.sass
  • Pour eviter de relancer la commende a chaque modification, on peut ajouter --watch
    sass --wtch cheminVerFicherSource.scss
    par exemple ici : sass --watch ./scss/test.scss
  • On peut surveiller tout un dossier :
    sass --watch nomDossierSource/:nomDossierCible/
  • On peut choisir de minimifier notre compilation avec l'option "--style compressed"
    sass --watch nomDossierSource/:nomDossierCible/ --style compressed
